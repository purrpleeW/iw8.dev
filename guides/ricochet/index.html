Ricochet Anti-Cheat

On versions 1.52 and up, Ricochet is required for the game to run correctly. If Ricochet Anti-Cheat is not running, the game can close without warning, or on later versions, not run at all.

Warning: Even if Ricochet is running on your device, you CANNOT get banned using our mod.
