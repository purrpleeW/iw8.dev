FAQ

¿How to play with a controller?

The mods don't change how controllers work, so every game supports them natively.
Unless your controller chip wasn’t supported when the build was released (Usually DualSense)
in that case, look for a DS4 tutorial on the net.

¿How to play with friends?

Go to the #lan-method channel. Vanguard & Modern Warfare/Warzone (1.41.2/1.60.0/1.64.5) supported.

¿How do i open the console?

Press the key under ESC, this will usually be tilde (` or ~) or ² on AZERTY keyboards.
If you have a 60% keyboard you should press FN and ESC instead.

¿How do i change my username?

Use the command: name
(Lifix is the template, use the name that you want)

¿How can i see more camos?

You can see all the camouflages by pressing "Camo Swap" on the bottom of the camouflages menu.
You can also create custom ones with the docs in this site.

¿What is the best version?

You need to choose that by yourself 😊

¿How do i change my field of view?

You can change it on the game settings, or with cg_fov and a custom value (65-120)
for applying a multiplier to the current value use cg_fovscale and add a custom value (0,5 to 2)

¿How can i spawn in WZ with loadouts?

Use this command before match: set scr_br_loadout_option standardammo;set scr_br_allowLoadout 1;set scr_br_use_set_loadouts 1;set scr_br_allowLoadoutOnlyInPreLobby 0
To disable this, invert all the numeric values from it.
