FAQ

Note: If you have any of these questions, please read these before making a ticket in our Discord server!

How do I play with a controller?

The mods don't change how controllers work, so every game supports them natively. Unless your controller chip wasn't supported when the build was released (usually DualSense), look for a DS4 tutorial online.

How do I play with friends?

Go to the #lan-method channel on Discord. Vanguard & Modern Warfare/Warzone (1.41.2 / 1.60.0 / 1.64.5) are supported.

How do I open the console?

Press the key under ESC. This is usually tilde (` or ~) or ² on AZERTY keyboards. If you have a 60% keyboard, press FN + ESC instead.

How do I change my username?

Use the command: name Lifix
Replace Lifix with the username you want.

How can I see more camos?

You can see all available camouflages by pressing Camo Swap at the bottom of the camouflage menu. You can also create your own using the documentation available on this site.

What is the best version?

You'll need to decide that for yourself!

How can I play with bots?

Currently, bots require scripts. You can find them in the #community-extras channel.

Bots (GSC) by RyGuy2Cool provides regular game bots.
Bot Overhaul GSC by cilism includes more advanced AI behaviour.

How do I change my field of view?

You can change it through the in-game settings or by using:

cg_fov — sets a custom FOV value (65-120)
cg_fovscale — applies a multiplier to your current FOV (0.5-2.0)

How can I spawn in Warzone with loadouts?

Run the following command before starting a match:

set scr_br_loadout_option standardammo; set scr_br_allowLoadout 1; set scr_br_use_set_loadouts 1; set scr_br_allowLoadoutOnlyInPreLobby 0

To disable it, invert the numeric values.
