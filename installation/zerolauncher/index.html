Using ZeroLauncher

This guide covers installing and launching IW8-Mod with ZeroLauncher. This installation method is highly recommended for an easy setup experience.

Recommended: ZeroLauncher is usually the easiest option for standard users.

What ZeroLauncher Does

  • Launches the game without Python
  • Attempts to foolproof some issues that can occur during the manual setup.
  • Provides you with game files without needing to extract from a zip.
  • User-friendly

Setup Steps

  1. Download ZeroLauncher.
  2. Place it wherever you want, preferably C:\Games\ZeroLauncher
  3. Open ZeroLauncher.exe
  4. Select Modern Warfare and your desired game version from the dropdown.
  5. Press download
  6. Select the installation path for the game, ensure it's in a folder such as C:\Games\ModernWarfare
  7. Wait for the game to download, once done, press 'Launch iw8-mod'.
    (The launcher should automatically download our mod into the game folder for you.)

Troubleshooting

  • 'Driver is not installed' -> Reboot your PC.
  • Incorrect install path
  • Missing files or permissions
