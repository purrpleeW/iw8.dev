Manual Installation

This guide covers installing IW8-Mod manually. This method is only recommended for those who have trouble using our launcher.

Note: Manual installation is usually best for advanced users, testing, and troubleshooting specific file issues.

Requirements

  • A supported copy of the game (list in our about section).
  • The required IW8-Mod DLL files.
  • Python (installed via the website, NOT the microsoft store).
  • Administrator access (only used for python).

Installation Steps

  1. Download your MW2019 build (game files) from a trusted source.
  2. Download the IW8-Mod files from our Discord server.
  3. Copy the required mod files into the game folder.
  4. Copy _start_game_.py into the game folder.
  5. Launch the game by double-clicking the Python script, ensure you have python installed from the website.

Verify the Install

Once installed correctly, the game should load as normal but with our modifications. These include our small watermark in the bottom-right where the build number is, and in-game console and more.

Common Issues

  • Incorrect folder placement (eg. not in C:\Games\ModernWarfare).
  • Missing dependencies such as python.
  • Unsupported game version (contact us if you find a build that isn't supported).
