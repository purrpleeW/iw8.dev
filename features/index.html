Overview

IW8-Mod is a community-focused Modern Warfare (2019) client, which allows for modding capabilities not possible within the regular game.

Features list (Public Patch 2.3)

  • In-game developer console
  • GSC Scripting and injection support
  • Customn detailed crash handler (backported from MWII).
  • Global loadout + setting saving
  • Map file checking - (Maps are now able to be filtered so you don't accidentally load a map that doesn't exist.)
  • Increased global bot limit - (All modes now have a standard bot limit of 150, no need to use cheat-tables or similar to achieve that.)
  • Reverse camos, use any camo on any gun. Just use the command reverse_camos on if on patch 2.3. (If on patch 3, use the dedicated button to toggle this in the camo tab)
  • Global loadout + setting saving
