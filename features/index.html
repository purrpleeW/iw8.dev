Overview

IW8-Mod is a community-focused Modern Warfare (2019) client, which allows for modding capabilities not possible within the regular game.

Features list

  • In-game developer console.
  • Custom detailed crash handler (backported from MWII).
  • Global loadout & settings saving.
  • Map file checking.
  • Increased global bot limit.
  • Dedicated button for "reverse camos".
  • Custom images & animated images support.
  • LAN (P2P) functionality, play with friends!
  • GSC scripting with hot reloads, auto compiles upon launching.
  • Online menus.
  • "Matchmaking" (without emulation)
  • Discord RPC & invites functionality
  • In-game text chat.
  • Custom localized strings support.
  • Custom LUA support.
  • Voice chat in lobby.
  • More to come... 😼
